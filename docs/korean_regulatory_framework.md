# 한국 법제도와 연결된 최소노출형 사전검토 workflow

본 문서는 본 연구의 BM25 minimal_text 조건을 한국 전략물자 수출통제 체계와 연결하여, "정보 최소 노출 하에서의 사전 후보검토"를 제도적으로 설명한다.

## 1. 관련 한국 법제 요약

### 1.1 대외무역법
- **목적**: 수출입 및 해외투자의 공정한 거래질서 확립, 무역의 자유와 균형 있는 국제수지 유지
- **전략물자 관련 조항**: 제18조(전략물자등 수출의 제한), 제19조(전략물자등 수출승인 등)
- **핵심**: 전략물자 수출 시 사전 승인·신고·자기판정 등의 절차를 거쳐야 한다.
- **출처**: [법제처 대외무역법](https://www.law.go.kr/LSW/lsInfoP.do?lsiSeq=241530&lsNm=%EB%8C%80%ED%99%98%EB%AC%B4%EC%97%85%EB%B2%95#0000)

### 1.2 전략물자 수출입고시
- **목적**: 대외무역법 제18조에 따른 전략물자등의 수출입 기준과 절차를 구체적으로 규정
- **핵심**: 
  - 전략물자 수출 시 수출신고를 하여야 하며, 신고서에는 물품명, 규격, 용도, 최종소비자 등을 기재
  - 신고 내용이 사실과 다르거나 누락 시 과태료·취소 등의 제재
- **업무 흐름**: 수출자 → (수출신고) → 산업통상자원부 장관 → (심사) → 승인/거부
- **출처**: [전략물자 수출입고시](https://www.law.go.kr/LSW/admRulInfoP.do?admRulSeq=2100000270104&chrClsCd=010201)

### 1.3 산업기술보호법
- **목적**: 국가핵심기술과 산업기술의 유출 방지, 기술보호 체계 구축
- **핵심**: 해외 기업·기관과 기술협력 시 기술이전 계약서에 기술보호 조항을 포함하는 등 기술유출 방지 조치를 강구하여야 함
- **관련**: 전략물자 수출 신고 시 기술자료가 함께 이전되는 경우, 산업기술보호법상 '해외 유출'로 평가될 수 있음
- **출처**: [산업기술보호법](https://www.law.go.kr/LSW/lsInfoP.do?lsiSeq=247501&lsNm=%EC%82%B0%EC%97%85%EA%B8%B0%EC%88%A0%EB%B3%B4%ED%98%B8%EB%B2%95#0000)

### 1.4 국가핵심기술 제도
- **목적**: 국가경쟁력·안보에 핵심적인 기술의 해외 유출 방지
- **핵심**: '국가핵심기술'로 지정된 기술은 해외 이전 시 별도 승인 필요
- **운영**: 산업통상자원부·국가안보실·방위사업청 등이 협의하여 지정·관리
- **참고**: 국가핵심기술의 해외 이전은 산업기술보호법과 대외무역법을 동시에 적용받을 수 있음
- **출처**: [국가핵심기술 제도](https://kaits.or.kr/web/content.do?menu_cd=000067)

## 2. 최소노출형 사전검토 workflow 제안

본 연구의 minimal_text 조건과 BM25 검색 성능은, 아래 workflow에서 "사전 단계 보조 도구"로 활용될 수 있다.

```
기업 담당자
   │
   ▼
(Step 1) 제품·기술 설명을 시스템에 입력
   → "리튬이온 배터리 양극재 제조장비를 인도 업체에 수출"
   → 입력 정보는 내부 서버에만 저장, 외부 AI/검색 서비스에 전송하지 않음
   │
   ▼
(Step 2) BM25 minimal_text 조건으로 공개 통제목록 후보검색
   → 반환: 통제번호 + 핵심 설명 1문장 (예: "1C350 | Lithium and compounds...")
   → 전체 원문은 반환하지 않음
   │
   ▼
(Step 3) 후보 목록 확인 → 관련 항목이 있는 경우 YesTrade 자가평가 또는 전문평가로 이동
   → 최종 판정: 공식 절차 (YesTrade / 관세사 / 산업통상자원부)
   │
   ▼
(Step 4) 신고서 작성 시 필요 최소한의 정보만 사용
   → 통제번호, 품목명, 규격, 용도, 최종소비자
   → 기술자료(도면, 특허 등)는 별도 산업기술보호법 검토 필요
```

## 3. 법제도적 기여와 한계

### 기여
- **정보최소화 원칙**: 현재 한국 수출통제 신고 절차는 대부분 오프라인 또는 전용 시스템 내에서 진행되지만, 외부 검색 서비스 활용이 확대되는 상황에서 "어떤 정보까지 외부에 전송해도 되는가"에 대한 기준점을 제시한다.
- **사전검색 보조**: BM25 minimal_text는 YesTrade 자가평가 전 단계에서 "어떤 통제번호가 관련 있을 수 있는지"를 빠르게 식별하는 1차 필터로 사용할 수 있다.
- **법적 책임 분리**: 본 시스템이 후보를 제시하는 수준까지 기능을 제한하면, 최종 판정은 공식 절차에 맡기므로 법적 책임 소재가 명확해진다.

### 한계
- BM25-only baseline은 어휘 수준 매칭이라 전문 용어·동의어 커버리지가 제한된다. 후속 연구로 Dense retrieval을 결합할 필요가 있다.
- 합성 벤치마크와 상담형 모사 질의셋의 결과가 0/30로 극히 낮아, 실제 기업 질의 일반화는 추가 검증이 필요하다.
- 본 workflow는 사전검색 보조일 뿐, 수출 신고 승인을 대체하지 않는다.
- 국가핵심기술 지정 여부는 별도로 확인해야 하며, 본 시스템의 검색 결과로 판단하지 않는다. 본 연구는 국가핵심기술을 자동 판단하지 않는다.
